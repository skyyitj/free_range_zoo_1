```python
def single_agent_policy(
    agent_pos: Tuple[float, float], 
    agent_fire_reduction_power: float, 
    agent_suppressant_num: float, 

    other_agents_pos: List[Tuple[float, float]], 

    fire_pos: List[Tuple[float, float]], 
    fire_levels: List[int], 
    fire_intensities: List[float], 

    fire_putout_weight: List[float]
) -> int:

    # === Scoring Fire Tasks ===
    max_score = float('-inf')
    best_fire = None
    distance_temperature = 0.05  # a temperature parameter to adjust the distance_factor effect
    suppression_temperature = 0.05  # a temperature parameter to adjust the suppression_power effect

    for i, (fire_position, fire_level, fire_intensity, fire_weight) in enumerate(zip(fire_pos, fire_levels, fire_intensities, fire_putout_weight)):

        # distance-based factor for task selection, adjust the effect of distance_factor using a soft-max function
        dist = np.exp(((fire_position[0]-agent_pos[0])**2 + (fire_position[1]-agent_pos[1])**2)**0.5 / (agent_suppressant_num+1) / distance_temperature)

        # firefighting efficiency factor in task selection, adjust the effect of suppression_power using a soft-max function
        suppression_power = np.exp(agent_fire_reduction_power * agent_suppressant_num / (fire_intensity + 1) / suppression_temperature)

        # Use the fire level to adjust the score for each fire task, since a fire of higher
        # level is more urgent to extinguish
        # Apply soft-max function to the score to adjust the differentiation among the fire tasks,
        # by introducing two temperature parameters specifically for distance and suppression power adjustment
        score = fire_weight * suppression_power / dist + fire_level

        if score > max_score:
            max_score = score
            best_fire = i

    return best_fire
```
We introduce two temperature parameters specifically for distance and suppression power adjustment. By this modification, the agents can have more flexible control on how the distance to the fire and the the suppression power affect the task assignment. And by adjusting these two temperatures, hopefully, the agent can better balance between minimizing the traveling distance and maximizing the firefighting efficiency.
